  • External loads: None (settlement-only case)

Theory (Partitioned Direct Stiffness Method):
  
  Partitioned DSM equations:
  ┌           ┐ ┌   ┐   ┌           ┐
  │ K_ff K_fr │ │u_f│ = │ F_f - K_fr·u_r │
  │ K_rf K_rr │ │u_r│   │ F_r - K_rr·u_r │
  └           ┘ └   ┘   └           ┘

  Where:
  • u_f = free DOF displacements (solved for)
  • u_r = restrained DOF values (prescribed settlements)
  • F_f = active external loads
  • K_fr·u_r = settlement-induced load correction

Settlement Computation:
  1. Identify restrained DOFs (boundary conditions)
  2. Apply prescribed displacement values: u_r = [-0.002 m]
  3. Compute correction: F_correction = K_fr @ u_r
  4. Modify RHS: F_modified = F_external - F_correction
  5. Solve for free DOFs
  6. Reconstruct full displacement vector with prescribed values
  7. Compute reactions from K @ u_full

Results Obtained:

  ✓ RHS correction applied during solve()
  ✓ Reactions induced by settlement computed
  ✓ Member end forces reflect settlement effects
  ✓ Equilibrium satisfied (ΣRy = 0 for zero external load)
