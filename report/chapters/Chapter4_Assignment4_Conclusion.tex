\chapter{Conclusion}

This study presented an extension of a 2D object-oriented structural analysis program based on the Direct Stiffness Method to include thermal loading and support settlement effects. Thermal actions were incorporated through equivalent nodal forces derived from fixed-end formulations, while support settlements were implemented via stiffness matrix partitioning with prescribed displacements. The extensions were designed to integrate with the existing architecture without altering the core solution workflow.

The developed solver was verified through unit, interface, and regression tests, and validated against benchmark results obtained from FTOOL/SAP2000. Excellent agreement was observed across nodal displacements, support reactions, and member-end forces, with discrepancies attributable only to numerical precision. The results confirm that the solver provides a consistent and reliable framework for linear static analysis of frame--truss systems under mechanical, thermal, and settlement effects.
