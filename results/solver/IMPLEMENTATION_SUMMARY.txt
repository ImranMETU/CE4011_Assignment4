Structure Settlement Computation
─────────────────────────────────
Location: Assignment3/q2_frame_analysis/model/structure.py

Added Helper Methods:
  • _apply_settlement_correction()
    Implements partitioned DSM correction: F_corrected = F - K_fr @ u_r
    Called in solve() before conjugate gradient solver

  • _mat_vec(a, x)
    Static helper for matrix-vector multiplication

Modified Methods:
  • solve()
    Now calls _apply_settlement_correction() before solving

  • full_displacement_vector()
    Includes prescribed settlement values for restrained DOFs

  • from_dict(data)
    Parses "prescribed_displacements" field from JSON input

Settlement Theory:
  Partitioned DSM with prescribed displacements:
  
  [K_ff  K_fr] [u_f]   [F_f - K_fr·u_r]
  [K_rf  K_rr] [u_r] = [F_r - K_rr·u_r]
  
  Where u_r is fixed (prescribed settlements)
  Solve for u_f, then compute reactions: R = K_rf @ u_f + K_rr @ u_r
